# CookBook - Кулинарная книга на Django

## 📖 Описание проекта
CookBook - это полнофункциональное веб-приложение для управления рецептами, созданное на Django 5.1.1. Позволяет пользователям создавать, редактировать и делиться кулинарными рецептами с удобной системой поиска и фильтрации.

## 🚀 Основные возможности

### 👨‍🍳 Управление рецептами
- **Создание рецептов**: детальное описание с инструкциями
- **Редактирование**: полный контроль над содержимым
- **Удаление**: с подтверждением действия
- **Изображения**: загрузка и отображение фото блюд
- **Категории**: организация по типам (завтраки, обеды, ужины)

### 📝 Система ингредиентов
- **Детальное описание**: количество и единицы измерения
- **Связи ManyToMany**: гибкая система связей рецепт-ингредиент
- **Промежуточная модель**: RecipeIngredient для хранения количества
- **Фильтрация**: поиск рецептов по конкретным ингредиентам

### 📚 Коллекции рецептов
- **Персональные коллекции**: организация рецептов по темам
- **Публичные коллекции**: возможность делиться подборками
- **AJAX интерфейс**: добавление в коллекции без перезагрузки
- **Модальные окна**: удобный выбор коллекций

### ⭐ Система оценок и комментариев
- **Рейтинги**: оценка рецептов от 1 до 5 звезд
- **Комментарии**: обсуждение рецептов
- **Автоматический расчет**: средний рейтинг
- **Отображение**: звездочки и числовые значения

### 🔍 Интеллектуальный поиск
- **Полнотекстовый поиск**: по названию, ингредиентам, автору
- **Регулярные выражения**: поиск полных слов
- **Фильтры**: по категориям, сложности, ингредиентам
- **AJAX**: динамические результаты без перезагрузки

### 👤 Система пользователей
- **Регистрация и авторизация**: стандартная Django система
- **Личные кабинеты**: управление рецептами и коллекциями
- **Права доступа**: редактирование только своих рецептов
- **Профили**: информация об авторах рецептов

## 🔧 Установка и запуск

### Локальная разработка
   ```bash
# Клонирование проекта
git clone <repository>
   cd CookBook

# Виртуальное окружение
   python -m venv .venv
source .venv/bin/activate  # Linux/Mac
# или
.venv\Scripts\activate     # Windows

# Установка зависимостей
   pip install -r requirements.txt

# Миграции базы данных
   python manage.py migrate

# Создание суперпользователя
python manage.py createsuperuser

# Запуск сервера
   python manage.py runserver
```

### Продакшн развертывание на PythonAnywhere

#### 1. Загрузка файлов
- Загрузите весь проект в папку `/home/viktor811/CookBook/`
- Убедитесь, что файл `viktor811_pythonanywhere_com_wsgi.py` находится в корне проекта

#### 2. Настройка виртуального окружения
```bash
cd /home/viktor811/CookBook
python3.10 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

#### 3. Настройка базы данных
```bash
python manage.py migrate
python manage.py createsuperuser
```

#### 4. Настройка статических файлов
```bash
python manage.py collectstatic --noinput
```

#### 5. Настройка WSGI файла
В панели PythonAnywhere:
- Перейдите в раздел "Web"
- В поле "WSGI configuration file" укажите путь к файлу:
  `/home/viktor811/CookBook/viktor811_pythonanywhere_com_wsgi.py`

#### 6. Настройка статических файлов в панели
Добавьте следующие маппинги:
- **URL:** `/static/` → **Directory:** `/home/viktor811/CookBook/static/`
- **URL:** `/media/` → **Directory:** `/home/viktor811/CookBook/media/`

#### 7. Перезапуск веб-приложения
После всех настроек нажмите кнопку "Reload" в панели PythonAnywhere
