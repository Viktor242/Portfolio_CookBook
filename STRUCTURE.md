# 🍳 Кулинарная книга - Django проект

## 📁 Структура проекта

```
CookBook/
├── static/                    # Статические файлы
│   ├── css/
│   │   └── main.css         # Простые стили
│   ├── js/
│   │   └── main.js         # Простой JavaScript
│   └── images/
│       └── logo.svg        # Простой логотип
├── media/                   # Медиа файлы (загружаемые пользователями)
│   └── recipes/
│       └── images/         # Фотографии рецептов
├── templates/               # HTML шаблоны
│   ├── main/
│   │   └── base.html       # Базовый шаблон
│   ├── recipes/
│   ├── users/
│   └── search/
├── config/                  # Настройки Django
│   ├── settings.py
│   └── urls.py
├── users/                   # Приложение пользователей
├── recipes/                 # Приложение рецептов
├── collections_app/         # Приложение коллекций
├── search/                  # Приложение поиска
└── db.sqlite3              # База данных
```

## 🎨 Статические файлы

### CSS (`static/css/main.css`)
- **Простые стили** без сложных эффектов
- **Минималистичный дизайн** с базовыми цветами
- **Обычные формы** без анимаций
- **Простая типографика** (Arial)
- **Базовые кнопки** и элементы

### JavaScript (`static/js/main.js`)
- **Простая функциональность** без сложных анимаций
- **Базовое добавление/удаление** ингредиентов и изображений
- **Простой поиск** с поддержкой Enter
- **Подтверждение удаления** через confirm()

### Изображения (`static/images/`)
- `logo.svg` - Простой логотип без градиентов

## 📸 Медиа файлы

### Фотографии рецептов (`media/recipes/images/`)
- Автоматически загружаются при создании рецептов
- Поддерживаются форматы: JPG, PNG, GIF
- Простое отображение без эффектов

## ⚙️ Настройки Django

### Статические файлы (`config/settings.py`)
```python
STATIC_URL = "static/"
STATICFILES_DIRS = [
    BASE_DIR / "static",  # Папка со статическими файлами проекта
]
STATIC_ROOT = BASE_DIR / "staticfiles"  # Для collectstatic
```

### Медиа файлы
```python
MEDIA_URL = "/media/"
MEDIA_ROOT = BASE_DIR / "media"
```

## 🚀 Запуск проекта

1. **Установка зависимостей:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Миграции базы данных:**
   ```bash
   python manage.py migrate
   ```

3. **Сбор статических файлов (для продакшена):**
   ```bash
   python manage.py collectstatic
   ```

4. **Запуск сервера разработки:**
   ```bash
   python manage.py runserver
   ```
